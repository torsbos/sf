# sf - simple fetch
a simple fetch script  written in bash
<img src="https://raw.githubusercontent.com/torsbos/sf/refs/heads/main/preview.jpg" alt="preview of sf"/>

<br>
## usage
make `sf` into an executable:
```
$ chmod +x sf
```

you can put this file in `/usr/local/bin` and invoke it in `.bashrc`/`.zshrc` etc.

you can also put this file anywhere and symlink it to `/usr/local/bin`

## dependencies
`bash`
